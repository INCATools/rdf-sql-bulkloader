# rdf-sql-bulkloader

Bulk load of SQL table from RDF in Python

```
CREATE TABLE statement (
	id TEXT,
	subject TEXT,
	predicate TEXT,
	object TEXT,
	value TEXT,
	datatype TEXT,
	language TEXT,
        graph TEXT
);
```
